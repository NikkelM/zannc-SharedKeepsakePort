# Shared Keepsake Port
A mod which ports over multiple keepsakes from Hades 1 to Hades 2.<br>
Currently, this mod contains Thanatos, Megaera, Sisyphus, Eurydice, Patroclus, Orpheus, Hermes, Persephone and Achilles, although to access Hermes and Persephone, you will need to install [KeepsakePort](https://thunderstore.io/c/hades-ii/p/zannc/KeepsakePort/) (this mod is automatically installed alongside KeepsakePort).<br>
>[!NOTE]
>The primary function for this mod is to be used as a dependency for [Zagreus' Journey](https://thunderstore.io/c/hades-ii/p/NikkelM/Zagreus_Journey/) and [KeepsakePort](https://thunderstore.io/c/hades-ii/p/zannc/KeepsakePort/).
